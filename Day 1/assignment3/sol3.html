Data types in Javascript:

1. Number: represents both integers and floating point numbers.

2. strings: these are series of characters written within quotes(can use both single or double quotes).

3. Booleans: these can only have two values: 'true' of 'false'. These are often used in conditional testing.

4. Arrays: these are written within square brackets ([]) with items seperated by commas. * Array indexes are zero-based.

5. Objects: these are written curly braces ({}). Object properties are written as 'name:value pairs', seperated by commas.

6. Undefined: these are variables without any value, i.e, has the value undefined. The type is also undefined.

7. Null: null refers to 'nothing', i.e, something that doesn't exist. The data type of null is object.
