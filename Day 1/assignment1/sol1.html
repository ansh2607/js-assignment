What does console do?

A web console is a tool mainly used to log information associated with a web page like: network requests, javascript, security errors, warnings, CSS, etc. It enables us to interact with the web page by executing javascript expression in the contents of the page.

Different methods in console function: (examples in js file)

* log(): used to print the output the console.

* error(): used to log error message to the console.

* warn(): used to log warning message to the console.

* clear(): used to clear the console.

* table(): used to generate a table inside the console. The inputs provided must be either arrays or objects.

* time() and timeEnd(): used to know the amount of time spent by a block or a function.

* count(): used to count the number that the function hit by this counting method.

* group() and groupEnd(): used to group contents in a seperate block.
