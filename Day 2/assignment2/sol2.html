Different Array and String methods: (Examples in js file)

Array methods:

1. toString() this method converts an array to a string array values

2. pop() and push(): pop() method removes the last element from an array and returns the value that was popped out.
push() method adds a new element to the end of an array and returns the new array length.

3. shift() and unshift(): these are shifting elements.
The shift() method removes the first array element and shifts all the other elements to a lower index and returns the string that was shifted out.
The unshift() method adds a new element to an array (at the beginning), and unshifts older elements and returns the new array length.

4. splice(): the splice() method can used to add new items to an array.
The "first parameter" defines the position where new elements should be added.
The "second parameter" defines how many elements should be removed.
The rest of the parameters define the new elements to be added.
This method returns an array with the deleted items.

5. concat(): this method creates a new array by merging existing arrays. It does not change the existing arrays, instead returns a new array.

6. slice() this method slices out a piece of an array into a new array. It can take two arguments. The method then selects elements from the start argument, and end up to (not including) the end argument.

7. forEach(): this method calls a function once for each element in an array, in order. It requires a value of the current element as one parameter.

8. map(): this method creates a new array with the results of calling a function for every array element.It calls the provided function once for each element in an array, in order. It requires a value of current element as one parameter.

9. sort(): sorts the elements of an array.

10. reverse(): reverses the order of the elements in an array.

String methods:

1. indexOf() and lastIndexOf(): indexOf() returns the position of the first found occurence of a specified value in a string.
lastIndexOf() returns the position of the last found occurence of a specified value in string.

2. endsWith(): checks whether a string ends with specified string/characters.

3. includes(): checkd whether a string contains the specified string/characters.

4. match(): searches a string for a match against a regular expression, and returns the matches.

5. repeat(): returns a new string with a specified number of copies of an existing string.

6. replace(): searches a string for a specified value, or a regular expression, and returns a new string where the specified values are replaced.

7. search(): searches a string for a specified value, or regular expression, and returns the position of the match.

8. startsWith(): checks whether a string starts with specified characters.

9. slice(): extracts a part of a string and returns a new string.

10. split(): splits a string into an array of substrings.

11. substring(): extracts the characters from a string, between two specified indices.

12. toLowerCase(): converts a string to lowercase letters.

13. toUpperCase(): converts a string to uppercase letters.

14. trim(): removes whitespaces from both ends of a string

15. concat(): joins two or more strings, and returns a new joined strings.
